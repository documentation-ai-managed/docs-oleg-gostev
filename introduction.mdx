## Overview

Oleg Gostev provides a centralized hub for your project documentation. You create, organize, and collaborate on docs using familiar Markdown and MDX syntax. Access everything from a clean interface designed for teams and solo developers alike.

## Key Features

Oleg Gostev offers powerful tools to streamline your documentation workflow.

<Columns cols={3}>
  <Card title="Rich Editing" icon="edit-3" href="#editing">

  </Card>
  <Card title="Version Control" icon="git-branch" href="#versions">

## Quick Start

Get up and running quickly with these steps.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` using your email.
  </Step>
  <Step title="New Space" icon="plus">
    Click **New Documentation Space** and name it "Oleg Gostev".
  </Step>
  <Step title="First Doc" icon="file-text">
    Add a new page with frontmatter and Markdown content.
  </Step>
</Steps>

<CodeGroup tabs="MDX,YAML">
  ```mdx
  ---
  title: My First Doc
  description: Hello world example.
  ---

  ## Welcome

  This is your first document.
  ```
  ```yaml
  title: My First Doc
  description: Hello world example.
  ```
</CodeGroup>
