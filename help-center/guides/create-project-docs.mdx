## Overview

Follow these steps to start a new documentation project with Oleg Gostev.

<Steps>
  <Step title="Log in">
    Go to [oleggostev.com/login](https://oleggostev.com/login) and sign in to your account.
  </Step>
  <Step title="Create a New Project">
    Click "New Project" and enter your project name and description.
  </Step>
  <Step title="Start Adding Content">
    Use the editor to add pages, code, and resources as needed.
  </Step>
  <Step title="Publish">
    Click "Publish" to make your documentation live for collaborators or the public.
  </Step>
</Steps>
