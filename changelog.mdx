## Migration Guide

Follow these steps to upgrade Oleg Gostev smoothly. Each version includes backward-compatible changes where possible, but review breaking updates.

<Steps>
  <Step title="Backup Your Data" icon="database">
    Export your documentation space:

````bash
ogostev export --space my-docs --format zip
````

  </Step>
  <Step title="Update Package" icon="download">
    Install the latest version:

    <CodeGroup tabs="npm,yarn">
````bash
npm install ogostev@latest
````
````bash
yarn add ogostev@latest
````
    </CodeGroup>
  </Step>
  <Step title="Run Migrations" icon="settings">
    Apply database migrations:

````bash
ogostev migrate --version 1.3.0
````

    For v1.1.0 breaking change, update your config:

````yaml
# Old config.yaml
features:
  mdx: true

# New config.yaml
features:
  components:
    mdx: true
    interactive: true
````
  </Step>
  <Step title="Verify Installation" icon="check-circle">
    Test your space:

````bash
ogostev serve --space my-docs
````

    Visit `http://localhost:3000` and check changelog renders correctly.
  </Step>
</Steps>

<Callout kind="tip">
  Subscribe to [GitHub releases](https://github.com/oleggostev/docs/ releases) for instant notifications on new versions.
</Callout>
